업데이트 내역

버전 변경 사항
1.5.0
  • 다음 카페 게시글 읽기 차단 우회 기능 추가.
  • 새 탭이 현재 탭의 바로 옆에 열리도록 수정.
  • 네이버 카페 스탭게시판 게시글 접근 오류 해결.
  • 자체 도메인이 적용된 네이버 카페의 게시글을 일반적인 방식으로는 접근하지 못할 때 그 대안을 알림창으로 출력하도록 개선.
  • 업데이트 내역을 직관적인 목록 형태로 변경.
1.4.1
  • 1.3.0 이하 버전에서 1.4.0 이상 버전으로 업데이트한 직후 일시적으로 팝업 창에 프로그램이 꺼진 상태로 표시되는 문제 해결.
  • 불필요한 프로그램 권한 제거.
1.4.0
  • 프로그램을 팝업 창에서 즉시 켜고 끌 수 있는 기능 추가.
  • 네이버 카페의 모바일 버전 웹페이지에서도 동작하도록 기능 개선.
  • 업데이트 후 알림 기능 추가.
  • 프로그램 아이콘 개선.
1.3.0
  • 게시글 페이지에서 이전 게시글과 다음 게시글로 이동할 수 없는 문제 해결.
  • 구글 크롬 브라우저 구버전(72 미만) 호환성 개선.
1.2.0
  • 구글 크롬 브라우저 최신 버전(72 이상)에서 새 탭이 무한히 생성되는 문제 해결.
1.1.0
  • 카페 내 특정 게시글이 새 탭에서 열리지 않고 현재 탭에서 열리며 접근이 차단되는 문제 해결.
1.0.0
  • 초기 버전.
